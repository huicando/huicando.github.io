####js的特点：
1.解释型语言
2.类似于C和java的语法结构
3.动态语言
4.基于原型的面向对象。

###第一天学习
控制浏览器弹出一个警告框

alert("....");

在页面中输出一个内容(可以向body中输出一个内容)：

doument.write()

向控制台输出一个内容:

console.log("...")

可以将js代码编写到外部js文件中，然后通过script标签引入，写到外部文件中可以在不同的页面中同时引用，也可以利用到浏览器的缓存机制，是推荐使用的方式。（在script中加入src="路径"）
script一旦用于引入外部的文件，里面在写代码就会失效，如果需要需要另外新写标签

####基本语法：
注释：

/*
    多行注释
    不会被执行，但是可以在源码中查看
*/
//单行写到后面才有效。

注意事项：
1.js严格区分大小写。
2.js中每一条语句以分号(;)结尾

如果不写分号，浏览器会自动添加，但是会消耗一些系统资源。而有时候，浏览器会加错，所有分号必须写。

3.js中会忽略多个空格和换行，所以我们可以利用空格和换行对代码进行格式化。

####字面量和变量：
字面量指一些不可改变的值（可以直接使用，但是一般都不会直接使用字母量）。
变量→变量可以用来保存字面量，而且变量的值是可以任意改变的，变量更加方便我们使用，所以在开发中都是通过变量去保存一个字面量。
声明变量:
js中使用var关键字来声明一个变量。声明后要对其进行赋值。
标识符
在js中所有的可以由我们自主命名的都可以称之为标识符（变量名，函数名，属性名等）
命名规则如下：

  1. 标识符中可以含有字母，数字，_，$
  2. 标识符不能以数字开头
  3. 标识符不能是ES中的关键字或保留字
  4. 标识符一般都采用驼峰命名法

    首字母小写，每个单词的开头字母大写，其余字母小写。

####数据类型：
共有六种：

  1. String字符串
  2. Number 数值
  3. Boolean 布尔值
  4. Null 空值
  5. Undefined 未定义
  6. Object 对象

其中1-5为基本数据类型，Object属于引用数据类型

#####String字符串：

  • 在js中字符串需要使用引号引用起来。
  • 使用双引号或单引号可以，但是不能混着用。
  • 引号不能嵌套，双引号不能放双引号，单引号不能放单引号（可以在单双混合使用）
  • 在字符串中我们可以使用 \ 作为转义符。

#####Number数值：
在js中所由的数值都是Number类型（包括整数和浮点数）
js中可以表示用Number.MAX_VALUE数值的最大值Number.MIN_VALUE表示0以上的最小值。

Infinity表示正无穷。
-Infinity表示负无穷。
NAN表示不是数值

注意：
js在整数的运算基本可以保证精确，如果使用js进行浮点运算，可能得到一个不精确的结果，使用千万不要使用js进行对精确的要求比较高的运算。

#####Boolean布尔值：
布尔值只有两个，主要用来做逻辑判断。
true：表示真，false：表示假。

#####Null 空值和Undefined 未定义
null值专门用来表示一个为空的对象，使用typeof检查一个null值会返回object。
Undefined类型的值只有一个，就是undefind当声明一个变量，但是并不给变量赋值，它的值就是undefined。

*可以使用typeof 检查变量类型，语法typeof 变量 *

####强制类型转换：
指将一个数据类型强制转换为其他数据类型。
类型转换主要指，将其他的数据类型，转化为String Number Boolean。

#####将其他数据类型转化为String
方式一：
调用被转化的数据类型的toString()方法。

该方法不会影响到原值，它会将转换的结果返回。
但是注意：null 和 undefined这两个值没有toString（）

方法二：
调用String()函数，并将被转换的数据作为参数传递给函数。

可以对null 和 undefined 进行使用。

#####将其他的数据类型转化为Number
方法一：
使用Numbe()函数
1.字符串–>数字

如果为纯数字，则将其转化为数字
如果字符串有非数字的内容，则转化为NaN
如果是空格或者是多个空格，则转化为0

2.布尔值–>数字

true –>1
false –>0

3.null–>数字 0
4.undefined –> NaN
方法二：
专门用在字符串
parseInt()/parseFloat()把字符串转化为整数或者浮点数。

遇到一个非数字就会停止。
原理：先把传入的值转化为字符串一个一个转化，如果传入的是布尔值，会转化成NaN

方法三：
可以使用基本运算符-进行转化。

原理:在使用减运算的时候，所有的数字都会转化为数字，所有可以直接用字符串减0就可以转换为数字。

方法四:
使用一元运算。在值前面加+进行转换。

####其他进制数：

  1. 在js中，如果需要表示16进制的数字，则需要以0x开头
  2. 如果需要表示8进制，需要以0开头
  3. 如果需要表示2进制，则需要以0b开头（不是所有浏览器都支持）

如果需要转化的时候，可以用parseInt(参数值，进制数)

####将其他的数据类型转化为Boolean
使用Boolean()函数
数字–>布尔值

除0和NaN,其余的都是true

字符串–>布尔值

除空值外，其余都是true

null和undefined都是false
